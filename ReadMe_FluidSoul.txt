Installation (Procreate on iPad)
	1.	Download the .brushset file to your iPad.
	2.	Tap it once.
	3.	It will automatically open in Procreate.
	4.	Look for the folder named “Fluid Soul – Volume 1.”

⸻

Usage Tips
	•	Use light-to-medium pressure for natural flow
	•	Best on canvas sizes around 3000–4000px
	•	Great for expressive lettering and textured depth
